# 🎨 AI-Powered Festival Advertising Tool

An AI tool that helps small business owners instantly generate **festival-themed marketing posters** without design skills.

---

## 🚨 Problem

Small businesses rely on festival marketing, but:

- ❌ No design skills  
- ❌ No time for creatives  
- ❌ Generic templates (Canva fatigue)  

Result: inconsistent and low-impact marketing during key festivals like Diwali, Eid, New Year.

---

## 💡 Solution

Generate **custom festival posters in seconds**:

- Enter business details  
- Select festival  
- Get AI-generated marketing poster instantly  
- Download or share  

No templates. No tools. No design knowledge.

---

## ✨ Key Features

- 🎨 AI-generated festival posters (unique outputs)  
- 🏪 Business-aware prompt generation  
- ⚡ Instant image creation  
- 📥 Download posters  
- 📤 Share via Web Share API  
- 🔁 Fallback support if AI fails  

---

## 🧠 Tech Stack

- React (Vite)  
- Puter.js (AI image generation)  
- CSS  

---

## ⚙️ How It Works

1. User enters business + festival info  
2. System builds AI prompt  
3. `puter.ai.txt2img()` generates poster  
4. User downloads or shares
